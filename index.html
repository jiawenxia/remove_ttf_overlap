字体去重叠工具

此工具可以处理 TrueType/OpenType 字体文件，移除字形中的重叠部分。
如何创建 Token?

您的 Token 不会被存储在任何服务器上，仅用于直接与 GitHub API 通信。

为什么需要 GitHub Token?

为了处理您的字体文件，我们需要在 GitHub Actions 上运行一个工作流。出于安全原因，我们需要您提供自己的 GitHub Token 来进行此操作。

如何获取 GitHub Token:

  1. 前往 GitHub Token 创建页面
  2. 如果需要，登录您的 GitHub 账户
  3. 确保选中 "repo" 和 "workflow" 权限
  4. 点击底部的 "Generate token" 按钮
  5. 复制生成的 token 并粘贴到上方的输入框中

注意: 您的 token 不会被保存到任何服务器，仅用于直接调用 GitHub API。
